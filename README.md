## Requirements

- **Node.js**: 20.15.0
- **Package Manager**: npm (comes with Node.js) or Yarn

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://gitlab.com/abhishek.trivedi/kammel-customer-user-flow.git
   cd <project-directory>
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

## Running the Project

### Development Server

To start the development server, run:

   ```bash
   npm run dev
   ```

This will launch the application at `http://localhost:5173`.

### Production Build

To create a production build, run:

   ```bash
   npm run build
   ```

The build output will be located in the `build` folder.

## Folder Structure

  - **public/**: Contains public assets and `index.html`
  - **build/**: Production build output (generated after running `npm run build`)
  - **src/**: Contains the source code of the application
  - **assets/**: Static assets like images and icons
  - **components/**: Reusable React components
  - **containers/**: Containers for different pages
  - **helpers/**: Utility functions and helper files
  - **navigator/**: Navigation routes
  - **pages/**: Page components for different routes
  - **store/**: Redux store

## Available Scripts

- `npm run dev`: Runs the app in development mode.
- `npm run build`: Builds the app for production.
- `npm test`: Runs tests.
- `npm run lint`: Runs the linter.

## Troubleshooting

- **Node version issues**: Make sure you are using Node.js 20.15.0. You can switch versions with tools like [nvm](https://github.com/nvm-sh/nvm).
- **Environment Variables**: Ensure that any required environment variables are set up. Create a `.env` file in the project root if necessary.
